Neo4j en 3 Pasos
Ago 21, 2013

No todo es SQL. De hecho existen unas joyas de bases de datos como Neo4j: NoSQL y orientada a grafos. Parece complicado pero no lo es; de hecho NOSQL es un reciente paradigma en el ámbito de las bases de datos. Para entender mejor que son bases de datos orientadas a grafos Ian Robinson (director de Neo Technology) lo explica en éste vídeo, luego la guía para instalar Neo4j.

Guía de instalación

Básicamente son tres pasos:

  1. Bajar la última versión de Neo4j.
  2. Extraer el contenido del archivo comprimido.
  3. Ejecutar el script correspondiente $directorio/de/neo4j/bin/neo4j start

¿Difícil? Verdad que no.

Instalando y configurando Octopress

Luego de años sin blog, debido a la perdida de MatandoTigres he decidido retornar con un espacio renovado. Esta semana, intentando abordar excelentes lenguajes como Ruby y auto-obligado a usar Git he decidido montar este nuevo Blog en Github usando una excelente herramienta desarrollada en Software Libre llamada Octopress. Que mejor manera de comenzar explicando como instalar y configurar Octopress, un framework de blogging para hackers. Manos a la obra entonces!
